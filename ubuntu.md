### COMMANDS
create a file 
> touch filename.type
>> touch sample.txt
> create a file in editor
>> vi filename.txt // gedit filename.txt

append to file
> cat > sample.txt

display the contents of a text file on the command line
> cat filename
>> cat sample.txt

move file to another directory
> mv [file] [directory]
>> mv sample.txt Desktop  

list files in a remote directory
> ls -l /some/directory
